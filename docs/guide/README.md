# Game Guides

Comprehensive strategy guides for Rise of Kingdoms events, alliance coordination, and commander progression.

## Overview

The Game Guides section provides detailed documentation for:
- **Event Guides** - Strategies for solo, alliance, and PvP events
- **Alliance Protocols** - Coordination procedures for guardians, rallies, and territory
- **Commander Strategy** - Progression paths and pairing recommendations

## Accessing Guides

Navigate to `/guide` in the app to access all guides. The guides are organized by category:

### Events (`/guide/events`)
- Solo events (Mightiest Governor, More Than Gems, etc.)
- Alliance events (Ark of Osiris, Alliance Mobilization)
- Co-op PvE (Ceroli Crisis, Ian's Ballads)
- PvP events (KvK, Territory Wars)
- Continuous events (Guardians, Holy Sites)

### Alliance (`/guide/alliance`)
- Guardian run schedules and protocols
- Rally procedures and callouts
- Territory management
- Alliance rules and conduct

### Commanders (`/guide/commanders`)
- F2P progression paths
- P2P optimization
- Commander pairings by game phase
- Talent builds

## Guide Features

Each guide includes:
- **Overview** - Quick summary of the event/topic
- **Mechanics** - How things work
- **Strategies** - Tips and approaches
- **Checklists** - Preparation and execution steps
- **Rewards** - What you can earn

## Contributing

Guides are maintained in the app's data files:
- `apps/web/app/guide/events/events-data.ts`
- `apps/web/app/guide/alliance/alliance-data.ts`

To add or update a guide, modify the appropriate data file and submit a PR.

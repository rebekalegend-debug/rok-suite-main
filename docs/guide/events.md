## Using Event Guides

Each event guide in the app includes:

1. **Overview** - What the event is about
2. **Requirements** - City Hall level, prerequisites
3. **Mechanics** - How scoring/rewards work
4. **Strategies** - General and alliance-specific tips
5. **Checklists** - Before, during, and after event tasks
6. **Rewards** - What you can earn

## Event Schedule

Check the in-game calendar for exact event times. Most events follow a predictable rotation:

- **Daily**: Sunset Canyon
- **Weekly**: Mightiest Governor, Lohar's Trial, Alliance Mobilization
- **Bi-weekly**: Wheel of Fortune, Karuak Ceremony
- **Monthly**: Golden Kingdom, Silk Road
- **Season-based**: KvK (every ~90 days)

## Tips for All Events

1. **Prepare resources** - Stock up on speedups, gems, and items before events
2. **Coordinate with alliance** - Many events reward alliance-wide participation
3. **Time your actions** - Some events have bonus periods or phases
4. **Track progress** - Use checklists to ensure you complete all objectives
5. **Prioritize rewards** - Focus on events that give you the most value
